# マージソート

## 概要

配列を分割しそれぞれをソートした後に，統合することでデータを整列させるアルゴリズム．これを再帰的に行い，全体のソートを行う．

## 計算量

- 最悪時間計算量: $O(n \log{n})$
- 平均時間計算量: $O(n \log{n})$
- 最良時間計算量: $O(n \log{n})$
- 最悪空間計算量: $O(n)$
- 安定性: 安定（実装による）

## アルゴリズム

1. 配列を分割する（通常は半分）
2. 分割された各部分配列に対して，再帰的にマージソートを行う．
3. 部分配列の要素が1つのみなら，再帰を終了する．
4. ソートの完了した各部分配列の先頭同士を比較し，順に結果を配列に格納する．
5. すべての部分配列がマージされると完了．
